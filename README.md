# Calculadora EBAC

Calculadora aritmética simples criada para o curso da EBAC.  
Permite realizar operações básicas (+, -, *, /) usando **dois inputs** e um **select**, com cálculo automático ao alterar os valores.

## Tecnologias
- Vue.js 3 (Options API)
- CSS Grid
- Vercel (deploy)
